Common issues and resolutions for the Hardened Hub.

Issues

🌐 Connectivity Issues

Site is unreachable

  • Check External IP: Verify your public IP matches Dynu.
  • Check Traefik: docker compose -f infra/core/docker-compose.yml ps

Bad Gateway (502/503)

The application is down or not on proxy-net.

🔒 SSL & Certificate Issues

If you see "Connection not private", Traefik may be failing the ACME challenge.

Dynu Quota Deadlock (Record Hoarding)

On the Dynu free tier (4-record limit), failed challenges can leave stale TXT records that block new attempts with a 503: Quota Exception.

Fix: Run the automated purge script to clear stale records and restart the challenge:

./scripts/purge-dynu.sh

Manual Inspection

# Check Traefik ACME logs
docker compose -f infra/core/docker-compose.yml logs -f traefik

# Run automated SSL diagnostic
./scripts/check-ssl.sh

🔐 Authentication (Authelia)

Stuck in Login Loop?

Sync server time: sudo ntpdate pool.ntp.org and clear cookies.

🐳 Docker & Performance

If disk is full, run cleanup:

docker system prune -a --volumes
